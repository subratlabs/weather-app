# Weather App

A Python weather application that displays real-time weather information using a weather API.

## Features

* Real-time weather updates
* Search weather by city
* Simple and user-friendly interface

## Technologies Used

* Python
* Tkinter
* Weather API

## How to Run

1. Install the required libraries.
2. Add your API key.
3. Run the Python file.
